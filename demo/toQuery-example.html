How it works
  • Fill in the form fields with example data
  • Use the buttons to produce a CONSTRUCT or SELECT query based on the current form state
  • The generated query includes value patterns for populated editors so it can be executed against external endpoints
  • toQuery() returns the query string only; running it is up to the hosting application
